# 暑期排班系統｜Shift 1 模組

A Plus 教學部暑期班（8/3 ～ 8/28）排班設定工具，Phase 1 版本。

🔗 **線上網址**：`https://你的帳號.github.io/你的repo名稱/`（部署後請自行補上）

---

## 功能概覽

- **教師管理**：新增／編輯教師資料
- **教室管理**：設定可用教室
- **時段設定**：
  - 拖曳排序調整時段順序
  - 依班型（Shift）套用預設時段組合
  - 每個時段可設定類型（正課／其他）、備注
  - 自動計算每個時段時數，時間錯誤會即時警示
- **營隊分組管理**：下午時段可依營隊拆分多個分組，方便安排人力
- **資料匯出／匯入**：可將目前設定匯出成 JSON 檔備份，或匯入還原

---

## 資料儲存方式（重要）

目前版本是**純前端靜態頁面**，所有資料儲存在使用者瀏覽器的 `localStorage` 中：

- ⚠️ **資料不會同步**：不同人、不同裝置、不同瀏覽器看到的資料是各自獨立的，並非共用同一份班表
- ⚠️ **清除瀏覽器資料會遺失設定**：建議定期使用「匯出」功能備份成 JSON 檔
- 若多人需要共同編輯同一份班表，需要接後端資料庫（例如 Firebase、Google Sheets API），屬於後續規劃項目

## 本機開發 / 修改方式

這是單一 HTML 檔案（`index.html`），包含所有 CSS 與 JavaScript，沒有額外依賴套件，直接用瀏覽器打開即可預覽，或直接編輯後上傳覆蓋即可更新網站內容。

---

## 部署方式（GitHub Pages）

1. 檔案需命名為 `index.html`（全小寫）
2. Repo 需為 Public
3. Settings → Pages → Source 選 `Deploy from a branch` → Branch 選 `main` → 資料夾選 `/ (root)`
4. 儲存後約 1–2 分鐘內生效
